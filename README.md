# note-apps-pages

Noteシリーズ各アプリの利用規約・プライバシーポリシー等を公開するGitHub Pagesリポジトリ。

- **PUBLICであることが正**：App Store審査に公開URLが必須で、無料GitHub PagesはPublicリポジトリ前提（全社gitルールの正当な例外。Private化禁止）
- 静的な法務ページのみを置く。アプリ本体のソース・機密情報は一切置かない（本体は各Privateリポジトリで隔離）
- 構成：`<アプリのslug>/privacy.html`（必要に応じて `terms.html`）

| アプリ | ページ |
|---|---|
| 3分日記Note | [3min-diary-note/privacy.html](3min-diary-note/privacy.html) |
